# Overview
A desktop app designed to help plant enthusiasts keep track of their flowers and daily care tasks. It lets users record flower details, watering schedules, and growth progress to ensure proper care and monitoring.

## Main Features
### 1. Flower Information Management
Stores basic details like flower name, species, color, and planting date.
<details>
<summary>CRUD:</summary>

• Create: Add new flower profiles.

• Read: View flower details.

• Update: Modify information such as growth notes.

• Delete: Remove old or dead flower entries.
</details>

### 2. Watering and Maintenance Schedule
Allows users to plan and log watering or maintenance activities.
<details>
<summary>CRUD:</summary>

• Create: Set watering and maintenance dates.

• Read: Display schedule logs.

• Update: Adjust frequency and schedule.

• Delete: Remove outdated tasks.
</details>

### 3. Growth Progress Tracker
Records flower growth details like blooming stage, color changes, and size updates.
<details>
<summary>CRUD:</summary>

• Create: Add growth logs.

• Read: View flower progress.

• Update: Add or edit notes.

• Delete: Remove incorrect entries.
</details>

### 4. Search and Filter Flowers
Allows users to search for flowers by name or species to view related information.
<details>
<summary>CRUD:</summary>

• Create: Add filters or tags and search options.

• Read: Display search results.

• Update: Modify search criteria.

• Delete: Clear filters/tags.
</details>

### 5. Reminder System (Optional)
Notifies users when it’s time to water or check on their flowers.
<details>
<summary>CRUD:</summary>

• Create: Set up automatic reminders.

• Read: Show upcoming reminders.

• Update: Mark completed reminders.

• Delete: Remove old ones.
</details>
